# Overview
Dealix is a decentralized platform leveraging NFTs to provide automated, trustless discounts on token purchases while enabling users to earn through a seamless affiliate program. Designed for transparency, security, and scalability, Dealix empowers holders, affiliates, and project owners to maximize value in the DeFi space.

# Features
NFT-powered discounts on live charts
Automated rewards distribution
Affiliate program for passive income
Multi-chain support (Ethereum and Base)
Secure smart contracts and transparent tracking
